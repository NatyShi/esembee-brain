# Content Creator Skill

## Purpose
Generates content for 3 brands (NatyShi, esembee, IPY Now) across multiple platforms.

## Trigger Words
"write", "post", "thread", "article", "blog", "newsletter", "content"

## Capabilities
- 11 pieces of content per trigger
- LinkedIn posts and articles
- X/Twitter posts and threads
- Blog articles (esembee only)
- Video scripts (YouTube, Instagram)
- Visual asset generation (headers, infographics)

## Quality Gates
- Voice/tone match per brand
- Platform correctness
- CTA clarity
- Hook effectiveness
- Visual assets included

## Output Format
Individual files per brand/platform in `Content/Generated/MM-YYYY/`
- File naming: `YYYY-MM-DD_Brand-Platform-Topic.md`
- Frontmatter with tracking fields
- Paste-ready format (no arrows, no long dashes)

## Brands
| Brand | Voice | Audience | Platform |
|-------|-------|----------|----------|
| NatyShi | Personal, direct, raw | Founders, builders | X (ES) + LinkedIn (EN) |
| esembee | Expert, framework-driven | SMBs, startups | LinkedIn (EN) + Blog |
| IPY Now | Results-oriented, show don't tell | Event organizers | X + LinkedIn + IG + YT |

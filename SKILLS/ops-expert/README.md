# Ops Expert Skill

## Purpose
Operations management: kanban, daily briefings, workflow optimization, metrics tracking.

## Trigger Words
"kanban", "briefing", "workflow", "task", "pending", "todo", "metrics", "review"

## Capabilities
- Kanban board management (Obsidian plugin)
- Morning briefing (9am): tactical pending focus, kanban status
- Evening wrap-up (7pm): completed tasks, idea triggers, carry-forward
- Workflow optimization recommendations
- Metrics tracking and alerts
- Idea Pantry management

## Quality Gates
- All tasks tracked in kanban
- No items lost between sessions
- Carry-forward clean (only valid items)
- Metrics up to date

## Output Format
- Briefings: Telegram-formatted (Markdown, no code blocks)
- Kanban: Obsidian Kanban plugin format
- Reports: Structured markdown
